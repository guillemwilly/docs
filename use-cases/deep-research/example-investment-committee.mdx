The [investment-team](https://github.com/agno-agi/investment-team) repo is a working deep-research system you can clone and run. Seven specialist analysts deploy a $10M equity portfolio across five orchestration architectures, grounded in a fund mandate and a research library, producing auditable memos and decisions that compound through shared learnings. It is every page in this section, shipped.

```bash
git clone https://github.com/agno-agi/investment-team && cd investment-team
cp example.env .env   # set GOOGLE_API_KEY and EXA_API_KEY
docker compose up -d --build
docker exec -it investment-team-api python -m app.load_knowledge
```

Then ask the Coordinate team *"Should we invest in NVIDIA?"* or run the workflow *"Run a full investment review on NVIDIA."*

## The cast

| Member | Model | Role |
|--------|-------|------|
| Market, Financial, Technical Analysts | Gemini 3 Flash | Specialist research with Exa and YFinance |
| Risk Officer | Gemini 3 Flash | Position sizing and mandate enforcement |
| Knowledge Agent | Gemini 3 Flash | RAG over the research library, memo navigation |
| Memo Writer | Gemini 3 Flash | Drafts the standardized memo to disk |
| Committee Chair | Gemini 3.1 Pro | The final BUY/HOLD/PASS decision, no tools |

It runs on Gemini, not the `gpt-5.5` used in this section's snippets. The pattern is model-independent; the flagship picked Gemini for cost and speed across seven agents.

## The pipeline

Market assessment runs first. Fundamental and technical analysis run in parallel as the "Deep Dive." Risk assessment weighs both. The Memo Writer composes the artifact. The Committee Chair commits to a decision with a dollar allocation. Every step is grounded in the same mandate, and every run can write a learning the next run reads.

## Adapt it

Swap the mandate, the analysts, and the research library and the same architecture researches anything that needs many perspectives synthesized into one defensible call: vendor selection, clinical review, underwriting, diligence. The committee shape is the product; the domain is configuration.
